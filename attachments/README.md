# Attachments Directory

Place your **resume** and **recommendation letter** PDF files in this folder.

The email system will automatically attach all `.pdf` and `.docx` files from this directory to every outgoing email.

## Expected Files
- `Calvin_Kinyanjui_Resume.pdf` (or similar)
- `Recommendation_Letter.pdf` (or similar)

## Notes
- Only `.pdf`, `.docx`, and `.doc` files are included as attachments.
- Other file types are ignored.
- File names are preserved as-is in the email attachments.
